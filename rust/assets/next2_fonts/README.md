# next2 fallback fonts

These fallback fonts are bundled for cross-platform Next2 danmaku glyph coverage.

- `NotoSansYi-Regular.ttf`
- `NotoSansGeorgian-Regular.ttf`
- `NotoSansLao-Regular.ttf`

Source: https://github.com/notofonts/noto-fonts
License: SIL Open Font License 1.1 (OFL-1.1)

These files are intentionally bundled so Next2 does not rely on OS-specific font paths.
